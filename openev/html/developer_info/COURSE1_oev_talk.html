Introduction

This document is based on notes from a course that Frank Warmerdam gave to new developers at Atlantis a few years ago. It provides an introduction to the major components of the development environment (Python, OpenGL, GTK, GDAL, PROJ.4, NumPy), the OpenEV system architecture, initial design objectives, and technical information about the data model, layer drawing, editing tools, user interface, and command-line interface.
